# 영어 단어 퀴즈 게임 

## 프로젝트 개요

이 프로젝트는 파이썬으로 개발한 **대화형 퀴즈 게임**입니다. 사용자는 영어 단어의 뜻을 맞추는 게임을 통해 어휘력을 향상시킬 수 있습니다. 프로그램을 종료하고 다시 실행해도 추가한 퀴즈와 최고 점수가 보존됩니다.

### 주요 특징
- **완전히 동작하는 퀴즈 게임** - 메뉴 기반 인터페이스
- **퀴즈 관리** - 퀴즈 추가, 조회, 삭제 기능
- **점수 추적** - 최고 점수 자동 저장 및 갱신
- **데이터 지속성** - JSON 기반 파일 저장/로드
- **예외 처리** - 모든 입력값 검증 및 오류 처리

## 퀴즈 주제와 선정 이유

### 주제: **영어 단어 (English Vocabulary)**

### 선정 이유
1. **학습 효과**: 일상에서 자주 사용되지 않는 고급 영어 단어를 학습할 수 있습니다.
2. **폭넓은 활용성**: 회사원, 학생, 수험생 등 다양한 사람들이 사용할 수 있습니다.
3. **점진적 난이도**: 기본 단어부터 고급 단어까지 포함할 수 있어 난이도 조절이 용이합니다.
4. **흥미로운 게임 요소**: 맞춤형 학습과 게임의 재미를 결합할 수 있습니다.
5. **확장성**: 주제별 퀴즈 세트(동의어, 반의어, 예문 등)로 쉽게 확장 가능합니다.

## 실행 방법

### 개발 요구사항 및 개발환경
- Python 3.10 버전 이상
- 추가 라이브러리 없음 (표준 라이브러리만 사용)
아래는 파이썬 버전 확인
```
python --version
Python 3.13.12
```
### 설치 및 실행

#### 1. 저장소 복제
```bash
git clone https://github.com/your-username/english-quiz-game.git
cd english-quiz-gam
```

#### 2. 프로그램 실행
```bash
python quiz_game.py
```

#### 3. 메뉴 사용
프로그램 실행 후 다음과 같은 메뉴가 표시됩니다:
```
==================================================
          영어 단어 퀴즈 게임
==================================================
1. 퀴즈 풀기
2. 퀴즈 추가
3. 퀴즈 목록 보기
4. 최고 점수 확인
5. 종료
==================================================
```

선택지 번호(1-5)를 입력하여 기능을 선택할 수 있습니다.
